비디오 태그

        video를 재생하는데 사용되는 태그
        controls : 재생, 일시정지, 음량조절 등의 기능을 활용할 수 있음
        autoplay : 자동재생
        muted : 비디오 음소거
        loop : 비디오가 끝나면 자동으로 반복해서 무한 재생
        source : 비디오 파일을 제공하기 위한 공간으로 실질적으로 비디오를 불러오는 공간

        사용예제
            video 태그 안에 source를 작성
            video 태그 옆에 controls를 작성

            비디오 태그 안에 source라는 태그 작성
                source 태그 옆에 src를 사용해서 비디오를 불러올 위치 지정
                    태그 옆에 type을 작성해서 비디오 형식 지정
