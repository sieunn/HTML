태그의 여백 관련 스타일

            1. padding 
                내용과 선 사이의 공간
            2. border
                영역이 어디까지 영역인지 구분하기 위한 선 (테두리)
            3. margin
                다른 태그와 간격을 나타내는 영역

padding margin border 방향성

            000 - padding , margin , border를 나타냄

            000-top : 크기      -> 위쪽 방향 크기지정
            000-bottom : 크기   -> 아래쪽 방향 크기 지정
            000-left :크기      -> 왼쪽 방향 크기 지정
            000-right : 크기    -> 오른쪽 방향 크기 지정

            단, border(선) 속성값은 [크기 모양 색상] 형식으로 지정
            예를 들어, 오른쪽 선은 도트모양으로 10px 빨간색
            border-right : 10px dotted red;

padding, margin, border-width 크기 한번에 지정하기

            padding | margin | border-width : 속성 값 크기 설정
            속성값 1개 : 상하좌우
            padding: 20px; /*상하좌우로 20px 주라는 의미*/
            margin: 20px; /*상하좌우로 20px 주라는 의미*/

            속성값 2개 : 상하 좌우
            padding: 20px 10px; /*상하 20px 좌우 10px */
            margin: 20px 10px; /*상하 20px 좌우 10px */

            속성값 3개 : 상 좌우 하
            padding: 20px 10px 30px; /*상 20px 좌우 10px 하 30px */
            margin: 20px 10px 30px; /*상 20px 좌우 10px 하 30px*/

            속성값 4개 : 상 우 하 좌 (시계방향)
            padding: 10px 20px 30px 40px; /*상 10px 우 20px 하 30px 좌 40px*/
            margin: 10px 20px 30px 40px; /*상 10px 우 20px 하 30px 좌 40px*/

margin : 0 auto;

            다른 태그와의 간격을 자동으로 지정해서 가운데 정렬하게 만드는 속성값

            기본적으로 자동 정렬 (= 가운데 정렬)
            display : block 값에서 자주 사용

            inline , inline-block은 text-aliga : center; 이용
